# analisis-de-datos

### 1. Crear de entorno virtual
```
    python3 -m venv .venv
```
### 2. Activar entorno virtual

windows (powershell)
```
    .\.venv\Scripts\Activate.ps1
```

windows (linux)
```
    source .venv/bin/activate
```


### 3. Instalar paquetes

``` 
    pip install -r requirements.txt
```
